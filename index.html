Using Mimo

Overall I can say that I like the app. It looks nice, it has a very intuitive(at least for me) navigation system, it is informative and easy.

Although, I am not sure so far if is going to actually help me to learn the language i need.

It feels like it does too much guiding and a lot of the stuff I don't know where to look up in case i need it. Of course you can goggle everything but still.

I am stil not done and have clearly learnt a lot but we will see at the end if it has any significant impact or if it is just a very nice way to start and I'll have to look for other sources.
